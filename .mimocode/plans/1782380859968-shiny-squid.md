# Plan: 课程颜色按周变化

## 目标
同一课程在不同周显示不同颜色。当前所有周的同一课程颜色相同（索引只基于课程名+教室），需要加入周次因素让颜色随周变化。

## 方案

### 核心改动：CourseColors.kt 加 `weekOffset` 参数

给 `getBackgroundStatic`、`getTextColor`、`getBackground` 加 `weekOffset: Int = 0` 参数，用 `(index/10 + weekOffset) % colors.size` 替代原来的 `(index/10)`，使同一课程在不同周偏移到调色板的不同色相。

### 调用链改动

1. **ScheduleApp.kt** — `courseColorMap` 计算时传入 `realCurrentWeek`
2. **WeeklyScheduleScreen.kt** — `blockBaseColors` 和文字颜色传入 `week`（当前页周次）
3. **TodayScreen.kt** — 今日页竖杠传入 `currentWeek`
4. **CourseDetailSheet** — 详情弹窗传入当前周次

### 涉及文件

| 文件 | 改动 |
|---|---|
| `app/src/main/java/com/classapp/schedule/util/CourseColors.kt` | 加 `weekOffset` 参数到 3 个公开函数 |
| `app/src/main/java/com/classapp/schedule/ScheduleApp.kt` | `courseColorMap` 计算传周次 |
| `app/src/main/java/com/classapp/schedule/ui/weekly/WeeklyScheduleScreen.kt` | blockBaseColors + textColor + DETAIL 传周次 |
| `app/src/main/java/com/classapp/schedule/ui/today/TodayScreen.kt` | 竖杠 + 考试竖杠传周次 |

### 效果
- 第17周周一第1节「英语」= 蓝色
- 第16周周一第1节「英语」= 偏移1个色相（如青色）
- 第18周周一第1节「英语」= 偏移2个色相（如绿色）
- 同一周内不同课程仍保持不同颜色

### 验证
- 构建安装后，翻到不同周查看同一课程颜色是否变化
- 今日页、详情弹窗颜色对应检查（日志对比）
